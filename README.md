# ERC721-ERC20Diamond
A Diamond standard contract that supports both non-fungible tokens (NFTs) and fungible tokens.

## Installation

- Clone this repo
- Install dependencies

```bash
$ yarn && forge update
```

### Compile

```bash
$ npx hardhat compile
```

## Deployment

### Hardhat

```bash
$ npx hardhat run scripts/deploy.js
```

### Foundry

```bash
$ forge t
```
